# MVVM-RxKotlin: Cryptocurrency app
This application has been made making use of the following technologies:
- Kotlin
- RxAndroid, RxKotlin
- Dagger2
- Retrofit
- GSON
- ViewModel
- Lifecycle-aware components
- DataBinding
- LiveData
